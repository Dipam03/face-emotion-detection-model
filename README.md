# Emotion Detection System 😃

This project detects human emotions using a trained deep learning model.

## 🔧 Tech Stack
- Python
- OpenCV
- TensorFlow 

## 📌 Features
- Real-time emotion detection
- Face detection using OpenCV

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the project:
python detector.py

## 📂 Files
- emotion_model.h5 → Trained model
- detector.py → Main code
